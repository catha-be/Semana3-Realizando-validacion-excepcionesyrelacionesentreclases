## 📘 Descripción general del sistema
Este proyecto corresponde a los avances de la Semana 3 de la asignatura *Desarrollo Orientado a Objetos I*. Se trata de un sistema orientado a objetos desarrollado en Java, cuyo objetivo es modelar y gestionar entidades representativas de un dominio, aplicando los principios de encapsulamiento, composición, herencia, polimorfismo e interfaces.

## 🧱 Estructura general del proyecto

```plaintext

    └── java/
        ├── app/   # Clase principal con el método main (Main.java)
        └── model/ # Clases de dominio (Empleado, Persona, etc.)
```

## ⚙️ Instrucciones para clonar y ejecutar el proyecto

1. Clona el repositorio desde GitHub:

```bash
git clone https://github.com/catha-be/Semana3-Realizando-validacion-excepcionesyrelacionesentreclases.git
```

2. Abre el proyecto (carpeta `Semana_3`) en tu entorno de desarrollo, como IntelliJ IDEA.

3. Dirígete a la clase `Main.java` ubicada en `src/main/java/app/Main.java`.

4. Ejecuta el archivo `Main.java`.

5. Sigue las instrucciones en consola o verifica la salida del programa.
